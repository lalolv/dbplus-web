# DB+ 管理界面

DB+ 项目的前端管理界面

## 安装向导

```shell
npm install
yarn
```

### 开发环境，调试编译运行

```shell
npm run serve
yarn run serve
```

### 生产环境，编译部署

```shell
npm run build
yarn build
```

编译部署后，放置到 Go 服务的 h5 目录下。

### 运行和测试

```shell
npm run test
```

### 检查语法

```shell
npm run lint
```
